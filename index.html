Welcome to ShapeKnots-GUI v0.0.2.

ShapeKnots is used to predict a set of structures that can contain pseudoknots. It can take SHAPE mapping data as a restraint to increase structure prediction accuracy.

ShapeKnots-GUI try to make easier the use of command line tool, putting a visual interface around it.
